# 1. Colleghiamo il tuo progetto locale al repository GitHub che hai creato

```bash
git remote add origin https://github.com/kmerlo/Stocks_Screener.git
```

# 2. Rinominiamo il branch in "main" (standard moderno di GitHub)

```bash
git branch -M main
```

# 3. Carichiamo i file (potrebbe aprirsi il browser per chiederti il login)

```bash
git push -u origin main
```

---

## Verification

The repository is now live and can be viewed at: <https://github.com/kmerlo/Stocks_Screener>

## Next Steps

You can continue working normally. To save future changes, just run:

```bash
git add .
git commit -m "descrizione della modifica"
git push
```
